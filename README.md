# CGEI

Protocolo de comunicação para aplicação de uma Estufa Inteligente.

Arquitetado para resolver problema de comunicação entre diferentes processos,
utilizando o conceito de sockets, que criam um caminho, ou abrem uma porta para
passagem de mensagens, dados, segmentos, pacotes, o que for necessário e restrito para a situação.

Realizado por alunos do ICMC, disciplina "SSC0142 - Redes de Computadores (2026)".
